# 理解逻辑门

逻辑门是数字电路的基本构建模块。它们对二进制输入执行基本逻辑操作以产生单个二进制输出。

## 基本逻辑门

### 与门（AND Gate）
- 仅当所有输入都为 1 时输出才为 1
- 真值表：0 与 0 = 0，1 与 1 = 1

### 或门（OR Gate）
- 当至少一个输入为 1 时输出为 1
- 真值表：0 或 0 = 0，1 或 1 = 1

### 非门（NOT Gate）
- 反转输入信号
- 真值表：非 0 = 1，非 1 = 0
